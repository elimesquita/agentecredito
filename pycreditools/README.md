# PyCrediTools Runtime

Versao enxuta do `pycreditools` mantida neste repositorio para uso pelo MCP de agente de credito.

O backup completo da biblioteca original fica em:

```text
../pycreditools_backup_20260610_1230
```

Esta pasta ativa mantem apenas os modulos necessarios para:

- simular politicas de credito;
- aplicar regras de producao;
- otimizar cortes de score;
- criar ratings de risco;
- exportar objetos usados pelo `mcp_server`.
